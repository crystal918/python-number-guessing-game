# Python Number Guessing Game 🎯

A simple command-line number guessing game built with Python.

## How It Works
The computer generates a random number between 1 and 100. The player guesses until they find the correct number. The game gives Too High or Too Low hints after each guess.

## Concepts Used
- Random number generation
- While loops
- If/elif/else statements
- User input handling

## How to Run
Type: python guessing_game.py in your terminal
